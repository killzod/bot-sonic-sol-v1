## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/killzod/bot-sonic-sol-v1.git
cd bot-sonic-sol-v1
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

buat file .env di root projek

```bash
SEED_PHRASE="your seed phrase here"
```

### 4. Run script

```bash
npm run start
```
